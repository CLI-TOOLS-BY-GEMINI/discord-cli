# 環境構築ガイド

本プロジェクトの開発および実行に必要な環境を整える手順を解説いたしますわ。

## 1. 動作要件
- **Go 1.26.1 以上**: 事前にインストールされていることをご確認くださいませ。

## 2. インストール手順
リポジトリをクローンした後、ルートディレクトリにて以下のコマンドを実行して、バイナリをビルドいたしますわ。

```bash
go build -o discord-cli ./cmd/discord-cli/main.go
```

## 3. 設定
Discord Bot トークンを環境変数として設定する必要がありますわ。

```bash
export DISCORD_TOKEN="あなたのBotトークン"
```

## 4. 開発とテスト
新しい機能を実装した後は、以下のコマンドでユニットテストを実行し、品質を確認することを推奨いたしますわ。

```bash
go test ./pkg/discord/... -v
```

## 5. 静的解析
コードの品質を保つため、`go fmt` や `go vet` によるチェックを適宜行うことをお勧めいたしますわ。
