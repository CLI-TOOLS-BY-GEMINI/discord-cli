# 技術スタック

本プロジェクトで使用されている主要な技術とバージョン、およびライブラリの情報ですわ。

## 言語とランタイム
- **Go 1.26.1**: 最新の Go 機能を活用し、高速で並行処理に優れた実行環境を提供しておりますわ。

## 標準ライブラリの活用
外部依存を最小限に抑え、Go の標準パッケージを最大限に活用しております。
- `net/http`: Discord API との通信に使用しておりますわ。
- `encoding/json`: JSON データのシリアライズ・デシリアライズを担います。
- `flag`: CLI 引数およびフラグの解析に使用しておりますわ。
- `testing`: 堅牢なテストスイートの構築に活用しております。

## 認証方式
- **Discord Bot Token**: `Authorization: Bot <TOKEN>` ヘッダーを用いた認証に対応しておりますわ。

## API バージョン
- **Discord REST API v10**: 現在の最新安定版バージョンに準拠しておりますわ。
